# meadowlark
Meadowlark is site travel agencies. Development with Node and Express.

# install and run
* `git clone https://github.com/DemyanchikVadim/meadowlark.git`
* `create yourself credentials.js`
* `npm install`
* `node meadowlark.js - for run app`
* `grunt - for run test`
* `http://localhost:3000`
